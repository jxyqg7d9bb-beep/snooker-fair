# 🎱 Snooker Fair - 遊戲計數結算工具

一個為棋牌遊戲（Snooker、Pot 波、啤珠等）設計的快速、準確的結算計算工具。支援兩種遊戲模式，自動計算每位玩家的水錢負擔、分數損益及最終結算金額。

## 功能特性

### 🎮 雙遊戲模式支援
- **Pot 波**：分數高者為贏家
- **啤珠**：分數低者為贏家

### 📊 完整的結算計算
1. **水數處理**：自動計算每位玩家的水錢負擔
2. **分數損益**：根據遊戲模式計算玩家之間的分數差額
3. **最終結算**：整合水錢和分數損益，得出每位玩家的最終應付/應收金額

### ✨ 用戶友善的介面
- 三步流程：設定 → 輸入 → 結果
- 實時計算，無需手動點擊計算按鈕
- 清晰的表格展示結算明細
- 支援複製結果到剪貼板
- 響應式設計，適配手機、平板、桌面

### 🛡️ 防錯設計
- 完整的輸入驗證
- 友善的錯誤提示
- 預設合理的初始值

## 使用流程

### 第 1 步：遊戲設定
1. 選擇遊戲模式（Pot 波 或 啤珠）
2. 設定遊戲人數（2-10 人）
3. 輸入分數倍數（每 1 分相當於多少金額）
4. 輸入水倍數（每 1 水相當於多少金額）
5. 輸入波鐘（遊戲開始時的總金額）

### 第 2 步：玩家資料
1. 輸入每位玩家的名稱
2. 輸入每位玩家的最終分數
3. 輸入每位玩家的水數

### 第 3 步：查看結果
- 系統自動計算並顯示：
  - 總水數、剩餘波鐘、每人平分金額
  - 各玩家的水錢負擔、分數損益、最終結算金額
  - 結算摘要（誰需要付多少、誰需要收多少）

## 計算邏輯

### 水數處理流程
```
1. 計算總水數 = 所有玩家水數之和
2. 計算剩餘波鐘 = 波鐘 - (總水數 × 水倍數)
3. 計算每人平分 = 剩餘波鐘 ÷ 玩家人數
4. 每位玩家的水錢負擔 = (個人水數 × 水倍數) + 每人平分
```

### 分數損益計算流程
```
對於每對玩家進行比較：
- Pot 波：分數高者贏，贏的金額 = |分數差| × 分數倍數
- 啤珠：分數低者贏，贏的金額 = |分數差| × 分數倍數
```

### 最終結算公式
```
每位玩家的最終結算 = 分數損益 - 水錢負擔
- 正數 = 該玩家需要收取的金額
- 負數 = 該玩家需要支付的金額
```

## 測試案例

### Pot 波 模式測試
**輸入**：
- 模式：Pot 波（高分贏）
- 人數：3 人（A、B、C）
- 分數倍數：5，水倍數：10，波鐘：190
- 分數：A=1, B=3, C=6
- 水數：A=4, B=3, C=6

**預期結果**：
- A：付 $95.00
- B：付 $55.00
- C：付 $40.00

### 啤珠 模式測試
**輸入**：同上，但模式為啤珠（低分贏）

**預期結果**：
- A：付 $25.00
- B：付 $45.00
- C：付 $120.00
